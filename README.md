Ore-Ore Language
================

A ultra primitive programming language that is a bit like Lisp

* [Demo](https://yamahei.github.io/orelang/index.html)

Overview
--------

![screen](/img/screen-min.png)

* JavaScript製オレオレ言語の実行環境
* Lisp風？文法
* FizzBuzzまでは動いた
* 構文木からフローチャート描画
  * [Viz.js](http://viz-js.com/)を利用

-----
